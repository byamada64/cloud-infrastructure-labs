# Azure Network Builds

Azure networking labs covering VNet design, subnet segmentation, NSGs, routing, private connectivity, and hybrid access patterns aligned to production architecture and security controls.

## Purpose

- Design: Evaluate segmented network architectures using VNets, subnets, and routing to control traffic flow, isolation, and service communication.

- Security: Review baseline controls including Network Security Groups (NSG), private endpoints, public exposure boundaries, and identity-aware access aligned to production hardening.

- Validation: Confirm connectivity, routing behavior, name resolution, and access control across public and private network paths.

- Limitations: Executed within a governed lab environment with constraints on advanced routing, firewall appliances, private DNS integration, and enterprise hybrid connectivity.
