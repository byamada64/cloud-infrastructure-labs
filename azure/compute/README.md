# Azure Compute Builds 

Infrastructure-based compute deployments using virtual machines and operating system-level workloads aligned to production design, access control, and operational considerations.

## Purpose

- Design: Evaluate infrastructure-first architectures requiring full OS control, custom configuration, and flexibility across workloads.

- Security: Review baseline controls including Network Security Groups (NSG), access methods (SSH/RDP), Public IP exposure, and identity/access considerations aligned to production hardening.

- Validation: Confirm deployment integrity, network connectivity, access configuration, and expected VM behavior across compute and networking layers.

- Limitations: Executed within a governed lab environment with constraints on advanced networking, identity integration, and enterprise-scale configurations.
